# 📊 Sales Data Modeling & Reporting Project

## 🚀 Project Overview

This project demonstrates an end-to-end sales data analytics workflow, starting from raw, unstructured data to building a structured data model and an interactive Excel dashboard for business insights.

The goal was to improve data accessibility, eliminate redundancy, and enable data-driven decision-making through efficient reporting.

---

## 🧱 Data Modeling

* Transformed raw sales data into normalized tables:

  * Customers
  * Products
  * Orders
  * Sales Transactions
* Defined **Primary Keys** and **Foreign Keys** to establish relationships
* Eliminated data redundancy and ensured data integrity
* Designed a scalable schema representing real-world business processes

---

## 🧹 Data Cleaning & Preparation

* Removed duplicate records and handled missing values
* Standardized:

  * Date formats
  * Product names
  * Pricing fields
* Ensured consistency and accuracy for reliable analysis

---

## 📊 Dashboard Development (Excel)

Built an interactive dashboard using:

* **Pivot Tables**
* **Charts**
* **Slicers**

### 🔑 Key Metrics (KPIs)

* Total Revenue
* Monthly Sales Trends
* Top-Selling Products
* Customer Segmentation

### 🎯 Features

* Dynamic filtering by City, Category, and Month
* Visual representation of trends and performance
* User-friendly and interactive design

---

## 📈 Business Insights

* Identified seasonal sales patterns and trends
* Highlighted high-performing products and categories
* Analyzed customer behavior and segmentation
* Supported data-driven decision-making
